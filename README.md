# UGUI-CircusFun

## 项目
基于AssetStore示例[Circus Fun](https://www.assetstore.unity3d.com/#!/content/28887)，使用UGUI全面改造。

## 更新
* 使用UGUI整合菜单界面、游戏界面、失败界面。
* 使用Shader更新背景贴图。
* 设计新的Prefab整合火圈、火坑、金币。
* 打包贴图，修复屏幕适配问题。
* 删除无用的刚体、碰撞器、触发器和声音源。
